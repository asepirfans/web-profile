  • HTML

    HTML (Hypertext Markup Language) adalah bahasa pemrograman yang digunakan untuk membuat halaman web. HTML digunakan untuk menentukan struktur dan konten suatu halaman web dengan menandai bagian-bagian penting dalam dokumen, seperti teks, gambar, dan tautan.

  • CSS

    CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengatur tampilan dan format halaman web. CSS digunakan untuk mengubah tampilan elemen HTML seperti font, warna, ukuran, dan posisi.

  • Javascript

    JavaScript adalah bahasa pemrograman yang digunakan untuk membuat interaksi pada halaman web. JavaScript dapat digunakan untuk membuat efek animasi, validasi formulir, pengolahan data, dan lain-lain.

  • IoT

    Internet of Things (IoT) memungkinkan pengumpulan dan analisis data dari berbagai perangkat untuk digunakan dalam berbagai aplikasi, seperti smart home, smart city, dan industri. IoT juga dapat memudahkan pengguna untuk mengontrol perangkat elektronik dari jarak jauh melalui koneksi internet.
